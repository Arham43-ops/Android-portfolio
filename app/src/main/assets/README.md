## ğŸ¨ Customization

### Colors
Edit CSS variables in `style.css`:
```css
:root {
    --accent-primary: #00d4ff;    /* Change accent color */
    --bg-primary: #0a0a0a;        /* Change background */
    --text-primary: #ffffff;      /* Change text color */
}
```

### Content
Update personal information in `index.html`:
- Change name, bio, and contact details
- Update project descriptions and links
- Modify skills and experience

### Images
Replace images in the `images/` folder:
- `img.jpg`: Your profile photo
- Project screenshots: Update with your actual projects
